# Lone Wolf: Kai Series - Game App

Desktop/Web application for playing the Lone Wolf Kai Series gamebooks by Joe Dever.

## Tech Stack

- **Tauri** - Desktop app framework
- **React + TypeScript** - UI
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **React Router v7** - Routing (Data mode)
- **Jotai** - State management
- **Motion** - Animations

## Getting Started

```bash
npm install         # Install dependencies
npm run dev         # Web dev server
npm run tauri dev   # Desktop app dev
npm run tauri build # Build desktop app
```
